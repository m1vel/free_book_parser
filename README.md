# _📚Парсер книг с открытой библиотеки Флибутса_

---

### _🌑P.S. Мой первый, объективно, полезный парсер для различной аудитории Но, увы, без графического интерфейса :(_

---

## _🚀Возможности:_
* ### Поиск книг по запросу
* ### Обработка всех страниц результа
* ### Сохранение данных в формате .xlsx
* ### Извлечение информации: автора, название и ссылка на скачивание
* ### Задержки в запросах для избежания блокировки

## _🛠️Установка:_

### _1. Клонировать репозиторий `(git clone )`_
### _2. Открыть папку склонированного репозитория `(cd <имя папки>)`_
### _3. Установить зависимости `(pip install -r requirements.txt)`_
### _4. Запустить файл add_to_excel.py `((python|python3) add_to_excel.py)`_

## _📦Зависимости:_
* #### requests
* #### beautifulsoup4
* #### lxml
* #### xlsxwriter

## _⚙️Пошаговая инструкция:_
#### _1. Запустите программу_
#### _2. Введите поисковой запрос_
#### _3. Введите название выходного файла_
#### _4. Дождитесь завершения_
#### _5. Найдите файл в папке с программой_
